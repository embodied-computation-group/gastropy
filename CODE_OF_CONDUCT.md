# Code of Conduct

This project adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. All contributors, maintainers, and participants are
expected to uphold this code.

## Reporting

Instances of unacceptable behaviour may be reported to the project maintainers
at **micah@cfin.au.dk**. All reports will be reviewed promptly and
confidentially.

## Attribution

This Code of Conduct is adapted from the
[Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
